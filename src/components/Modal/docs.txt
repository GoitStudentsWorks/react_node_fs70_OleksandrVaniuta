Як користуватися компонентом <Modal>

1. створити стейт який буде true або false
2. рендерити компонент <Modal/> де вам потрібно
3. передати пропси в <Modal/> перший значення вашого state. 
другий (якщо ви хочете закривати модалку по кліку на область)- функція зміни state в false.

приклад:  <Modal isActive={isActive} closeModal={closeModal}>
             <Ваша розмітка модалки>
          </Modal>


Тепер коли значення isActive === true модалка відкрита. Коли false закрита.
